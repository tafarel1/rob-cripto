# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [Unreleased]

### Adicionado
- Dashboard de Performance SMC (Estatísticas de acerto).
- Alertas Visuais (Pulso) para interação com zonas críticas SMC.
- Controles de Backtest (Replay, Play/Pause, Speed).
- Floating Controls para o gráfico SMC.
- Sistema de temas (Light/Dark) automático baseado no horário.

### Alterado
- Refatoração do `SMCMainChart` para melhor performance e modularidade.
- Melhorias na estrutura de arquivos do frontend.
- Atualização da documentação do projeto.

## [1.0.0] - 2024-05-20

### Inicial
- Versão inicial do Robô Cripto SMC.
- Integração com Binance e Bybit.
- Dashboard unificado.
- Sistema de análise SMC básico.
